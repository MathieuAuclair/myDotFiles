# myDotFiles
My configuration for i3Desktop with i3Gaps under Archlinux

to install my configuration, do the following steps
```
sudo pacman -S ttf-font-awesome-4 
sudo mv .lock /bin/lock
sudo mv .cool-rofi /bin/cool-rofi
reboot
```
